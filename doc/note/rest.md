# REST APIについて

## REST APIの操作種別

### POST
CREATE相当、サーバー側でリソースを割り当てる(AUTO_INC 等)

### GET
READ相当

### PUT
CRAETE or UPDATE、IDを明示的に指定する場合に使用する

### PATCH
UPDATE相当、一部の項目の更新で使用する

### DELETE
DELETE相当、リソースを削除する
